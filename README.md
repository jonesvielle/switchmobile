# Switch Pay DEMO

A secure React Native mobile banking app built for the Interswitch Frontend Mobile.

## Features

- Secure login with Account ID + PIN
- Biometric authentication
- Swipeable bank accounts dashboard
- Pull-to-refresh
- Animated drawer navigation
- Secure persistence with redux-persist + Keychain storage

## Tech Stack

- React Native (bare workflow)
- TypeScript
- Redux Toolkit + redux-persist
- react-native-biometrics
- React Navigation

## Setup

1. `npm install`
2. `cd ios && pod install && cd ..`
3. `npx react-native start --reset-cache`
4. Run: `npx react-native run-ios` or `run-android`
